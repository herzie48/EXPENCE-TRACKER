# 💰 Expense Tracker

A simple command-line Expense Tracker built using Python. This application allows users to record, view, remove, and calculate their expenses through an interactive menu.

## Features

* ➕ Add a new expense
* 🗑️ Remove an existing expense
* 📋 View all recorded expenses
* 💵 Calculate the total amount spent
* ✅ Input validation for removing expenses
* 📖 Simple menu-driven interface

## Technologies Used

* Python 3

## How to Run

1. Install Python 3 on your computer.
2. Clone or download this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python expense_tracker.py
```

## How to Use

1. Choose **Add an expense** to record a new expense.
2. Enter the following details:

   * Date
   * Category
   * Description
   * Amount
3. Choose **View all expenses** to display all recorded expenses.
4. Choose **Remove an expense** and enter the expense number to delete it.
5. Choose **Total expenses** to view the total amount spent.
6. Choose **Exit** to close the application.
